# Thailand's Exposure to Taiwan Semiconductor Supply Chain Disruption

## Research Question
How exposed are Thai industries to Taiwan semiconductor supply chain disruption, and which industries face the highest disruption impact?

## Overview
This independent research project analyzes Thailand's import dependency on Taiwan-originated semiconductors (HS Code 8542) from 2015 to 2024. The analysis identifies which Thai industries face the greatest risk in the event of a Taiwan semiconductor supply chain disruption.

## Methodology
- Trade data sourced from UN Comtrade Database
- Taiwan Dependency Index calculated as share of semiconductor imports from Taiwan vs. total
- Industry vulnerability scored based on import dependency and production value

## Tools & Technologies
- Python (pandas, plotly)
- Jupyter Notebook
- Data: UN Comtrade, OIE Thailand, TSMC Annual Report
