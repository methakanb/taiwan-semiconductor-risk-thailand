# Methodology Note: Taiwan Proxy in UN Comtrade/WITS

For this project, `Other Asia, nes` is used as a proxy for Taiwan/Chinese Taipei in Thailand's HS8542 import data.

This is not treated as a perfect one-to-one official identity. It is a proxy required by partner-country reporting limitations in the available UN Comtrade/WITS export. The report should therefore use careful language:

- Preferred: "Other Asia, nes (used as a Taiwan/Chinese Taipei proxy)"
- Avoid: "Taiwan" without qualification

Recommended limitation statement:

> Due to partner classification limitations in the available UN Comtrade/WITS dataset, Taiwan is not separately reported for Thailand's HS8542 import data. This study therefore uses `Other Asia, nes` as the closest available proxy for Taiwan/Chinese Taipei. Because this aggregate category may include minor flows from other unspecified Asian economies, the results should be interpreted as an estimate of Taiwan-related semiconductor exposure rather than an exact official Taiwan import value.
